# AWO Portfolio v2

Simple responsive HTML/CSS/JS portfolio with local SVG artwork included.

## Files
- index.html
- styles.css
- script.js
- assets/*.svg

## Run locally
Open `index.html` in a browser or serve the folder with any static server.

## Customize
- Replace `assets/awo-mark.svg` with your real AWO logo if you have it.
- Replace project SVGs with screenshots when ready.
- Update GitHub link in `index.html`.
